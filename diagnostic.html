How to Fix Missing PDF.js Files:

  1. Visit https://github.com/mozilla/pdf.js/releases
  2. Download the latest stable release (look for "pdfjs-x.y.z-dist.zip")
  3. Extract the zip file
  4. Copy "build/pdf.min.js" to your extension directory and rename it to "pdf.js"
  5. Copy "build/pdf.worker.min.js" to your extension directory and rename it to "pdf.worker.js"
  6. Refresh your extension in chrome://extensions/
